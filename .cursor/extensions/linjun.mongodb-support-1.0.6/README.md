# MongoDB

Support connecting to MongoDB in Visual Studio Code.

## Connect

1. Open Database Explorer panel, then click the `+` button.
2. Select your database type, input connection config then click the connect button.

![connection](https://doc.database-client.com/images/show.png)

## SQLite

For SQLite, you can open the db file and click the icon as shown to quickly connect.
